# Demis Hassabis 视角 Skill

> 用 DeepMind CEO & 诺贝尔奖得主 Demis Hassabis 的思维框架分析 AI 趋势、技术路线与科学战略。

---

## 优势长处

### 1. 真正的「思维框架蒸馏」，不是角色扮演

大多数 AI 角色扮演只是模仿语气和措辞，这个 skill 的核心是 **6 个可执行的心智模型**，每个都有来源证据、具体应用方法和已知局限：

- **信息第一性** — 看到指数级复杂问题时，先找背后的可学习低维流形
- **双轨推进** — 永远在规模化和算法创新两条路上分配资源
- **游戏沙盒思维** — 先在低成本环境模拟推演，再在现实中执行
- **爱因斯坦测试** — 用「能否独立提出相对论」判断真智能，不看基准分数
- **锯齿智能** — 知道 AI 在哪些维度超人类、哪些维度不如 5 岁小孩
- **科学优先** — 把 AI 当望远镜用，而不是当效率工具用

### 2. 先研究再判断，不编造事实

工作流强制要求搜索事实再回答，附带 5 种失败模式的兜底路径（搜索无结果/信息矛盾/超出领域/模型矛盾/信息过时）。不会凭空编造数据。

### 3. 诚实的边界标注

自动标注「哪些是事实、哪些是推断、哪些已超出可靠范围」。对 AGI 时间表等极端前沿话题，会显式降级为「逻辑外推，可靠性低」。

### 4. 哈萨比斯式表达

短句、类比、先结论后论证、用棋类和游戏比喻。不写废话、不做冗长的元分析、不贩卖焦虑也不画饼。

## 使用场景

| 场景 | 示例问题 | Skill 的核心价值 |
|------|---------|----------------|
| **AI 趋势分析** | Scaling laws 是否见顶？Mamba 能取代 Transformer 吗？ | 用双轨推进 + 信息第一性模型，基于最新事实而非空谈做判断 |
| **技术路线判断** | Google 的 AI 战略有效吗？Anthropic 为什么走安全路线？ | 用哈萨比斯的「第一性+长期主义」框架拆解，结合搜索到的真实数据 |
| **科学发现策略** | AI for Science 有哪些真机会？AlphaFold 之后下一个是什么？ | 用科学优先模型 + HiPPO 谱系分析，看真正的瓶颈在哪里 |
| **创业/投资决策** | 现在该投 AI 应用还是基础模型？没有算力怎么参与？ | 用锯齿智能模型分析能力边界 + 用沙盒思维设计低成本试错路径 |
| **长期战略思考** | 2030 年的 AI 格局会怎样？个人开发者怎么积累护城河？ | 用哈萨比斯的 10 年尺度 + 价值观优先级推演，不是预测具体事件而是判断结构性趋势 |
| **理解 Hassabis 本人** | 哈萨比斯为什么卖 DeepMind 给 Google？他怎么看待 AI 风险？ | 决策记录 5 个关键转折点 + 价值观体系 + 内在张力 |

## 提取过程

1. **信息收集** — 收集 Hassabis 近 5 年的核心演讲、Lex Fridman 访谈、学术演讲、YC 对话等一手素材，同时加入对传记《哈萨比斯：谷歌AI之脑》的提取和总结作为references。
2. **心智模型蒸馏** — 提取 6 个反复出现且可操作的心智模型，每个配来源证据 + 应用方法 + 局限
3. **表达 DNA 提取** — 分析其语言偏好（短句、类比、游戏比喻、先结论后论证）
4. **决策逻辑还原** — 还原 5 个关键决策时刻的思考逻辑
5. **价值观体系** — 提炼优先级排序的价值观 + 内在张力

## 优化过程

经 **Darwin Skill 2.0**（基于 Microsoft Research SkillLens + SkillOpt 论文方法论）进行 3 轮优化：

- **Round 1** — 添加失败模式三段式表 + CHECKPOINT 标记 + 输入输出规格 + 输出模板
- **Round 2** — 添加操作层黑名单 6 条 + 反模式强化
- **Round 3** — 补充中英文触发词

评分从 **64.6 → 79.5**（+14.9 分），所有改动经 git 版本控制，可随时回滚。
